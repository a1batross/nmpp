Подробное описание

Функция инициализации элементов массива постоянным значением.

. 

\f[
i = \overline{0 \ldots nSize-1}

\f]

Аргументы
nSizeРазмер вектора в элементах.
nValКонстанта. Диапазон значений nVal должен соответсвовать типу результирующего вектора.
Возвращаемые значения
pVecРезультирующий вектор.
Возвращает
void
