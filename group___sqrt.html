void nmppsSqrt_32f (const nm32f *pSrcVec, nm32f *pDstVec, nm32f *pTmpBuf, int nSize)
 

Подробное описание

Функция вычисляет квадратный корень от каждого элемента входного вектора чисел с плавающей точкой

Аргументы
[in]pSrcVecвходной вектор чисел с плавающей точкой
Возвращаемые значения
[out]pDstVec выходной вектор чисел с плавающей точкой
Аргументы
[in]pTmpBufдополнительный буфер, необходимый для ускорения работы функции (размер равен размеру pDstVec). Ускорение произойдет только при условии, что pDstVec и pTmpBuf лежат в разных банках.
[in]nSizeчисло элементов в векторе Функция совместима ТОЛЬКО с NMC-GCC-SDK и входит в состав библиотеки libnmpp-nmc4f.a
