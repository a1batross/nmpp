Подробное описание

Уменьшение высоты изображения вдвое.

. 

\internal

Значение пикселя для выходного изображения получаются в результате усреднения значений для соседних пикселей входного изображения.

Аргументы
pSrcImg
Исходное изображение.
pTmpBuf
Временный буфер размера 2*nSrcWidth*nSrcHeight.
nSrcWidth
Ширина изображения;
nSrcHeight
Высота изображения.
Возвращаемые значения
pDst
Результирующее изображение;


Restrictions:
Высота должна быть четным числом.
Заметки
Для того чтобы избежать переполнения при усреднении, динамический диапазон исходного изображения должен принадлежать диапазону, определенному типом.
